ENDPOINTS DE LA API

  • 'api/escultores/' para ver la lista de Escultores
  • 'api/eventos/' para ver lista de Eventos
  • 'api/eventos/'id'' para ver descripcion de evento
  • 'api/obras/' para ver la lista de Obras
  • 'api/obras/'id'' para ver descripcion de obra
  • 'api/imagenes/'
  • 'api/imagenes/'id''
  • 'login' para loguearse, espera un user y pass en JSON
  • 'register' para registrarse, el JSON es medio raro
  • 'api/votar_obra/'id'/' para votar

Cabe aclarar que todas las rutas sirven para cualquiera un GET, pero para POST, PUT, DELETE, tenes que ser super usuario...

JSON PARA REGISTER:

{ "user": { "username": "inserteuser", "first_name": "insertenombre", "last_name": "inserteapellido", "email": "example@mail.com", "password": "insertepass" }, "user_extra": { "fecha_nacimiento": "1990-01-01", "pais": "Argentina" } }
